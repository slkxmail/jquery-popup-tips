Плагин всплывающих подсказок.

Плагин показывает всплывающие подсказки. Подсказки можно привязать к любому объекту на странице или окну браузера и задать позиционирование относительно этого элемента или окна. Появление подсказок можно связать с любым событием на странице или временем прошедшем с загрузки страницы или появления подсказки. Подсказки можно выстраивать в цепочки таким образом что они будут появляться одна за другой в заданной последовательности.

Параметры всплывающей подсказки

  • showType - условие появления подсказки [time, event, now]:
    • time - по времени. Появление происходит через время указанное в параметре timeDelay.
    • event - по событию. Появление подсказки происходит по наступлению события указанному в параметре event.
    • now - подсказка отобразится немедлено
  • linked - Относительно чего отображать подсказку [window, object, cursor]
    • window - относительно окна
    • object - относительно объекта указанного в linkedObject
    • cursor - относительно курсора (при выборе этого параметра нельзя указывать координаты появления подсказки)
  • linkedObject - объект относительно которого отображается подсказка ( при условии что в поле linked указано значение object)
  • event - событие произошедшее у объекта заданного в параметре eventObject по которому появляется подсказка [blur, focus, load, resize, scroll, unload, beforeunload, click, dblclick, mousedown, mouseup, mousemove, mouseover, mouseout, mouseenter, mouseleave, change, select, submit, keydown, keypress, keyup]
  • eventObject - объект у которого отслеживаются события (заданные в параметре event) по наступлению которого надо показать подсказку.
  • content - содержимое окна подсказки
  • childs - дочерние подсказки
  • top - если linked:
    • window - отступ от верхнего края страницы
    • object - отступ вверх от центра объекта linkedObject
    • cursor - игнорируется
  • bottom - (игнорируется при заданно значении параметра top). Если linked:
    • window - отступ от нижнего края страницы
    • object - отступ вниз от центра объекта linkedObject
    • cursor - игнорируется
  • left - если linked:
    • window - отступ от левогокрая страницы
    • object - отступ влево от центра объекта linkedObject
    • cursor - игнорируется
  • right - (игнорируется при заданно значении параметра left). Если linked:
    • window - отступ от правого края страницы
    • objec - отступ вправо от центра объекта linkedObject
    • cursor - игнорируется
  • width - шарина окна подсказки
  • height - высота окна подсказки
  • onBeforShow - перед открытием подсказки
  • onAfterShow - после открытия подсказки
  • onBeforClose - перед закрытием подсказки
  • onAfterClose - после закрытия подсказки
  • singleton - подсказка выводится один раз
  • mask - фон под подсказкой. Если нет задано фон не выводится
    • color - цвет фона
    • opacity - прозрачность
